
How to run getting and cleaning project :
==================================================================

1 ] download or clone the project called Getting and cleaning data project in my github repository 

2 ] open Rstudio and run run_analysis.R file in this folder

3 ] A new file called tidy_data.txt will be created in the root folder directory .

4 ] you can display the tidy data by writing : tidy_data command in the console.

5 ] You can get the variables name of tidy data by writing : names(tidy data) in the console.
